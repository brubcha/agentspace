# Marketing Agent Client

This folder will contain the React (TypeScript) frontend for the Marketing Agent Tool.

To initialize the app, run:

```
npx create-react-app . --template typescript
```

Or, if you want to manually set up the project, install the following dependencies after initializing:

- @mui/material
- @mui/icons-material
- @emotion/react
- @emotion/styled
- file-saver
- docx

You can remove this file after setup.
