# Fonts Integration Guide

This folder is for custom font files (e.g., .ttf, .woff) used in the marketing kit generator.

## How to Use

1. Place your font files here (e.g., Inter.ttf, Roboto-Regular.woff).
2. Update your docx generation logic to reference these fonts for headings, paragraphs, etc.
3. Document font usage and mapping in implementation-docs/figma-to-docx-style-mapping.md.

## Example

- Inter.ttf: Used for all headings
- Roboto-Regular.woff: Used for body text

## Next Steps

- Ensure your docx export logic supports custom font embedding or referencing.
- Add font mapping details to your style mapping documentation.
